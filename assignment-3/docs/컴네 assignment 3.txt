docker 관련 : 
1. assign2에서 proxy 이름 중복으로 삭제 : 
docker rm -f proxy
+) 이름 확인 원하면 : 
docker ps -a

2. assign2와 같은 방식으로 생성.

3. docker container 안에서 다음 명령어 입력 :
apt install libnsl-dev


cd network/assign3

4. 일단 server용 1개 : 
docker start -i proxy
cd project3
./server

5. client용 1개 :
docker exec -it proxy /bin/bash
cd project3

- 디버깅용 
To enable packet logging, change the LOG_PACKET value to TRUE in the 18th line of the stcp_api.c file.

tcp_sum -> directly 사용 안 함 -> stcp_api에서 #include 해서 checksum 관련 이거 확인하면 될 것으로 보임.
